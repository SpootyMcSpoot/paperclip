# plugins

Staple plugin surface: SDK, scaffolder, and examples. Plugins extend the server with new routes, UI panels, skills, and background workers without forking core.

## Contents

- [`sdk/`](sdk/README.md) — `@stapleai/plugin-sdk`, the typed plugin contract
- [`create-staple-plugin/`](create-staple-plugin/README.md) — scaffolder (`npm create staple-plugin`)
- `examples/` — reference plugins:
  - `plugin-hello-world-example/`
  - `plugin-file-browser-example/`
  - `plugin-kitchen-sink-example/`
  - `plugin-authoring-smoke-example/`

See `doc/plugins/` for the authoring guide.

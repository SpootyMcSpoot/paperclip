# skills

Skill bundles shipped with Staple. Skills are runtime-injected prompts, workflows, and file templates that teach agents how to work inside Staple.

## Bundles

- `staple/` — core Staple skill (how to use tickets, heartbeats, governance)
- `staple-create-agent/` — guided flow for creating a new agent
- `staple-create-plugin/` — guided flow for scaffolding a plugin
- `para-memory-files/` — PARA-style memory file layout

See `doc/plugins/` and `packages/plugins/` for how skills are distributed.

# docs

Published documentation site (Mintlify). Hosted at <https://staple.ing/docs>.

## Layout

- `start/` — install, quickstart, first company
- `guides/` — task-oriented guides
- `api/` — REST API reference
- `cli/` — CLI reference
- `adapters/` — adapter catalog and configuration
- `deploy/` — deployment targets and modes
- `specs/` — product and protocol specs
- `plans/` — planning documents
- `images/`, `favicon.svg` — assets
- `docs.json` — Mintlify config
- `DEPLOYMENT-WORKFLOW.md`, `INTEGRATION_SERVICES.md`, `agents-runtime.md` — standalone references
